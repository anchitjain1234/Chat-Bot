mybot



mybot is a chat bot built on the Hubot framework.
It was initially generated by generator-hubot, and configured to be deployed on Heroku to get up and run as quick as possible.
To see it in action please click https://bitspilani8.campfirenow.com/ff6d5 and join the chat room.
Type mybot help to see a list of available commands .

Now included Pandorabot with mybot.
To try with improved bot try mybot pb query .

Some commands:-
1. mybot 9gag - See a random 9gag meme.
2. mybot animate me "query" :- Seraches a GIF for the given query.
3. mybot map me "location" :- Searches Google Maps for the given location.
4. mybot horsify me "query" :- Adds horse to the face of the image searched from the given query.
5. mybot annoy me :- To start getting annoyed from mybot.
6. mybot unannoy me :- To stop bot from annoying.
7. mybot simpsons image me :- Get a random Simpsons meme.
8. mybot simpsons quote me :- Get a random Simpsons quote.
9. mybot weather in "location" :- Get weather forecast for the specified location.
10. mybot qr me "url" :- Get QR code for the specified URL.
11. mybot email address@example.com -s "subject" -m "message" :- To send email using mybot.
12. mybot what is query :- To know about query.
13. mybot question "question" :- To get answer from wolfram alpha for your question.
14. mybot tvshow "show":- To get info about entered tv show.
